Introduction

This project probably will explore how messages can undergo transformation as they pass from one person to another. Through a combination of visual and auditory elements, I aim to create an engaging and thought-provoking experience.

Inspiration: In the Megaphone Game, teams consist of multiple members. The first person in each team receives a question and must convey its message to their teammates using only sound effects and gestures. They then pass the message to the next teammate in line, and this continues until it reaches the last teammate, who must guess the question based on the conveyed actions and sounds.
